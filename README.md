# EcoTeach — Green Flag Sustainability Analytics Platform

**MSc Group Project | Maynooth University | Business Analytics 2024–2025**

---

## 🌱 Project Overview

EcoTeach is a data-driven micro-credential platform developed as part of the MSc Business Analytics programme at Maynooth University. Designed to help Irish primary school teachers deliver sustainability education and support their schools in achieving **Green Flag certification** through the Green-Schools Programme (An Taisce).

**Phase 1 — Analytics Live:** Python data analysis, R statistical analysis, Power BI dashboards, Tableau real-time dashboard, and Moodle course design

**Phase 2 — Commercialisation and Biodiversity:** Addition of Biodiversity module, advanced Python analysis (BERTopic, Prophet), gamification, commercialisation planning, and platform refinement. Two groups collaborated on this phase.

---

## 📊 Data Analysis

### Datasets Used

- Irish Annual Energy Statement (SEAI)
- Global Data on Sustainable Energy (Kaggle)
- Green Energy Dataset (Kaggle)
- Electricity Generated by Fuel Type
- Ireland–Germany CO₂ Comparison
- Insulation Level and CO₂ Emissions data

### Phase 1 — Python (Jupyter Notebooks)

- Data cleaning and standardisation using **Pandas**
- Exploratory analysis and visualisations using **Matplotlib**
- Analysed energy trends, CO₂ outputs, and building efficiency correlations

### Phase 1 — R Analysis (ANOVA)

- Statistical analysis of insulation levels vs energy demand
- Statistical analysis of CO₂ emission sources
- Both tests returned p-values below 0.05 — statistically significant results
- Outputs: boxplots and bar charts used directly in course materials and dashboards

### Phase 2 — Advanced Python Analysis

- **BERTopic** — topic modelling on 1,600+ sustainability education case studies
- **Prophet** — time-series forecasting of day-ahead electricity prices

### Tableau — Real-Time Dashboard (Phase 1)

- Connected to Google Forms for live teacher submissions
- Tracks sustainability initiatives and CO₂ reduction metrics across schools in real time

### Phase 2 Dashboards

- **Course Tracking Dashboard** — tracks enrolments, completions, quiz scores, badges and certificates
- **Energy Impact Dashboard** — kWh saved, cost savings (€), and CO₂ emissions reduced
